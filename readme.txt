نمایش ویدئوهای دیدئو در مطالب با استفاده از شورت کد

== Description ==

با استفاده از این افزونه می توانید ویدئو های دیدئو را به سادگی در سایت وردپرس خود قرار دهید

با نصب این افزونه، دکمه ای به ویرایشگر وردپرس اضافه می شود که با استفاده از آن به سادگی می‌توانید ویدئوهای دیدئو را در سایتتان نمایش دهید.

This plugin is used to play dideo videos in your posts.

In This plugin we use [dideo](https://dideo.ir "dideo") for showing video in wordpress site.

You can get more info about dideo in [link](https://www.dideo.ir/about "dideo about").

== Installation ==

1. به برگه 'افزونه‌ها' در مدیریت وردپرس رجوع کرده و بر روی 'افزودن' کلیک کنید
2. 'wp dideo' را جستجو کنید
